Install and use the Prepackaged Virtualbox VM Image

Before You Begin

  1. Install VirtualBox by going to the following location: VirtualBox Downloads.
  2. Download the prepackaged appliance from: this link. (mirror 1, mirror 2) (update with new image)
  3. Download and install PuTTY
  4. Download and install WinSCP

ALTERNATIVE VIDEO TUTORIAL

Import the Appliance

1. Installation Guide

 

Download the prepackaged appliance from this link. (mirror 1, mirror 2)

To import the appliance, select the File > Import Appliance... menu option.

Click "Open appliance…"

Select the file from the downloaded location.

Hit the "Next Button"

Click "Import"

 

Wait as the appliance is imported

 

 

You should now have an image named "Chef Ubuntu 32 Bit". You can start this image by clicking the start icon at the top of VirtualBox.

A window should pop up and end up looking like this after the server boots. It is a 32-bit installation of Ubuntu 10.10 Server.

 

Feel free to log in with

Username: root

Password: school2012

 at this screen. However, SSHing into the server allows easier copy-paste, changing of the window size, and scrolling to view history. To do this, we must find the IP to connect via SSH.

Configure Putty

Open the Settings menu for the Chef Ubuntu VM

Open the Port Forwarding menu, and note the Host IP in Rule 1. In my case, it is 192.168.56.1

Open PuTTY and put the IP address from before into the Host Name field. Then Press "Open"

 

This message will appear the first time. Click Yes

Login as

root

school2012

 

And you will be greeted with this screen to indicate success!

Try typing "ls" and then hit enter.

 

Configure WinSCP

PuTTY is great for running commands, but if you want to copy files directly and are using Windows, I recommend WinSCP. Open WinSCP and click the New button.

Enter the IP into Host name, and the

User name: root

Password: school2012

These are the same as logging in with PuTTY

Click "Login".

(In future runs you may want to save)

Success! Now you can drag files between the locations, and even sychronize directories!

 

If you will log in frequently to edit files, click "Save…" on the previous screen. Then check the "Save password (not recommended)" box and click "OK"
